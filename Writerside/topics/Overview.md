## What is Microtus?
Microtus is a fork of the Minestom project that is based on [patches](Patch-Files.md). 

## Goals
- Include patches from open pull requests on Minestom repository
- Fixes issues from original Minestom repository
- Make useful decisions
  - Include new features
  - Try to stay updated with the original minecraft cycle
- Release cycle of 2 weeks
