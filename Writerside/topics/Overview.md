# Microtus

Hey and welcome to the documentation over the Project Microtus.
This wiki contains only the change which are made to the project. When you need more information about Minestom itself
please visit their dedicated [wiki](https://wiki.minestom.net/).
If you want to contribute to the project please read the [contribution guidelines](Contributing.md).

## What is Microtus?
Microtus is a fork of the Minestom project that is based on [patches](Patch-Files.md). 

## Goals
- Include patches from open pull requests on minestom repository
- Fixes issues from original minestom repository
- Make useful decisions
  - Include new features
  - Try to stay updated with original minecraft cycle
- Release cycle of 2 weeks
